**Super Market Billing System**

**Overview**
The Super Market Billing System is a Java Web Application developed using Servlets, JSP, and JDBC. The application allows supermarket staff to manage billing records efficiently by adding, viewing, and listing customer purchase details stored in a database.

**This project demonstrates practical implementation of:**
MVC Architecture
Servlets & JSP integration
Database connectivity using JDBC
Form validation and error handling
Record search and retrieval

**Features**
1. Add Bill Record
Users can add a new billing record including:
Customer Name
Item Name
Purchase Date
Quantity
Price
Remarks

**Validation Rules:**
Customer name cannot be empty
Item name cannot be empty
Quantity must be greater than 0
Price must be greater than 0
Duplicate records are not allowed

**2. View Specific Bill**
Users can search a bill using:
Customer Name
Purchase Date
Displays bill details if found, otherwise shows an error message.

**3. View All Bills**
Displays all stored billing records in tabular format. If no records exist, a "No records available" message is shown.

**Technologies Used**
Java (Servlets & JSP)
JDBC
MySQL / Oracle Database
Apache Tomcat Server
